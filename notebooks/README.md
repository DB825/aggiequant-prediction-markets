# notebooks/

- `01_walkthrough.py` - step-by-step demo of every stage of the
  pipeline on a small synthetic dataset. Runs as a plain script
  (`python notebooks/01_walkthrough.py`) or opens as cells in
  VS Code / Jupyter via the `# %%` markers.
- `02_kalshi_research_notebook.py` - portfolio-facing research
  notebook for Kalshi resolved-market ingestion, feature audit,
  walk-forward evaluation, Pareto-front model selection, and next-step
  findings across Prosperity-style microstructure, data engineering,
  sentiment, and ML extensions.
